# お手軽タスク管理アプリ v1.5 仕様書

## 1. バージョン概要

v1.5では、v1で実装した基本タスク管理機能を日常利用しやすくするための便利機能を追加する。

主な目的は、操作性、振り返り、検索性、日報作成の強化である。

## 2. v1.5の位置付け

v1は最小実用版である。

v1.5は、ユーザーが毎日使い続けられるレベルまで引き上げる実用強化版である。

## 3. v1.5で追加する主な機能

| 機能 | 内容 |
|---|---|
| グローバルショートカット | キー操作でアプリを表示する |
| 完了済み折りたたみ | 完了済みタスクを折りたためる |
| タスク検索 | タスク名で検索できる |
| 完了履歴 | 日付別に完了タスクを確認できる |
| 今日の作業時間詳細 | 完了タスクごとの時間を表示する |
| 簡易日報コピー | 今日の完了タスクをMarkdown形式でコピーする |
| タスク編集 | 既存タスクのタイトルを編集する |
| 起動時自動開始 | OS起動時にアプリを自動起動できる |
| 通知 | 長時間未完了タスクを通知する |
| 簡易メモ | タスクに短いメモを追加できる |

## 4. 前提条件

v1.5は、v1の以下機能が実装済みであることを前提とする。

- アプリ常駐
- ワンタッチ表示
- タスク追加
- 完了チェック
- 作業時間入力
- ドラッグ並び替え
- ローカル保存
- 今日の完了件数と作業時間合計表示

## 5. グローバルショートカット仕様

### 5.1 目的

マウスを使わず、キーボード操作だけでタスクリストを開けるようにする。

### 5.2 初期ショートカット

Windowsの場合。

```text
Ctrl + Shift + T
```

Mac対応する場合。

```text
Command + Shift + T
```

### 5.3 挙動

- アプリ非表示時は表示する
- アプリ表示中は前面に出す
- 入力欄に自動フォーカスする
- タスク入力欄へすぐ入力できる状態にする

### 5.4 ショートカット設定変更

v1.5では任意機能とする。

設定変更を実装する場合は、設定画面でショートカットキーを変更できるようにする。

## 6. 完了済み折りたたみ仕様

### 6.1 目的

未完了タスクに集中しやすくする。

### 6.2 表示例

折りたたみ状態。

```text
完了済みタスク 5件 [開く]
```

展開状態。

```text
完了済みタスク 5件 [閉じる]

☑ メール返信　10分
☑ 資料確認　20分
```

### 6.3 初期状態

完了済みタスクは折りたたみ状態を推奨する。

### 6.4 状態保存

ユーザーが開閉した状態は、アプリ再起動後も保持してよい。

ただし必須ではない。

## 7. タスク検索仕様

### 7.1 目的

過去タスクや現在のタスクをすばやく探せるようにする。

### 7.2 検索対象

- 未完了タスク
- 完了済みタスク
- 完了履歴
- メモ

### 7.3 検索条件

- タスクタイトルの部分一致
- メモの部分一致
- 大文字小文字は区別しない
- 入力中にリアルタイム検索する

### 7.4 画面例

```text
[ タスクを検索 ]

検索結果：
□ 仕様書を確認する
☑ 仕様書レビュー　30分
```

### 7.5 検索結果の表示

- 未完了タスクを上に表示する
- 完了済みタスクは完了日時を表示する
- 一致部分を強調表示してもよい

## 8. 完了履歴仕様

### 8.1 目的

日ごとに完了したタスクを振り返れるようにする。

### 8.2 表示例

```text
2026/05/28
- 仕様書を確認する　30分
- メール返信　10分
- 会議メモ整理　20分

合計：3件 / 1時間
```

### 8.3 表示単位

v1.5では以下を実装する。

- 今日
- 昨日
- 過去7日

任意の日付指定は将来拡張でよい。

### 8.4 集計項目

- 完了件数
- 作業時間合計
- 作業時間が最も長いタスク
- 作業時間未入力のタスク数

## 9. 今日の作業時間詳細仕様

### 9.1 目的

今日どの作業にどれだけ時間を使ったか確認できるようにする。

### 9.2 表示例

```text
今日の作業時間

- 仕様書を確認する：30分
- メール返信：10分
- 会議メモ整理：20分

合計：1時間
```

### 9.3 表示位置

以下のいずれかに配置する。

- メイン画面下部
- 履歴画面
- 詳細表示パネル

推奨は、履歴画面に詳細表示する方式である。

## 10. 簡易日報コピー仕様

### 10.1 目的

完了したタスクをMarkdown形式でコピーし、日報やメモに貼り付けられるようにする。

### 10.2 出力例

```markdown
## 2026/05/28 作業ログ

### 完了タスク

- 仕様書を確認する：30分
- メール返信：10分
- 会議メモ整理：20分

### 合計

- 完了件数：3件
- 作業時間：1時間
```

### 10.3 操作

メイン画面または履歴画面に以下ボタンを配置する。

```text
[今日の日報をコピー]
```

クリックすると、Markdown形式のテキストをクリップボードにコピーする。

### 10.4 コピー完了表示

コピー完了後、以下のような簡易通知を表示する。

```text
日報をコピーしました
```

## 11. タスク編集仕様

### 11.1 編集方法

タスクタイトルをダブルクリックすると編集モードにする。

```text
□ [ 仕様書を確認する ]
```

### 11.2 保存方法

- Enterで保存
- Escでキャンセル
- フォーカスアウトで保存

### 11.3 バリデーション

- 空欄では保存しない
- 前後の空白は削除する
- 変更時に updated_at を更新する

## 12. 起動時自動開始仕様

### 12.1 目的

PC起動時に自動でアプリを常駐させる。

### 12.2 設定項目

設定画面に以下項目を追加する。

```text
☑ PC起動時に自動起動する
```

### 12.3 初期値

初期値は OFF とする。

理由は、ユーザーの許可なく常駐アプリ化すると不信感が出るためである。

### 12.4 挙動

- ONにするとOS起動時にアプリを起動する
- 起動後はメイン画面を表示せず、タスクトレイ常駐状態にする
- OFFにすると自動起動を解除する

## 13. 通知仕様

### 13.1 目的

長時間放置されているタスクを控えめに知らせる。

### 13.2 通知条件

以下の条件を満たすタスクがある場合に通知する。

- 未完了
- 作成から一定時間が経過
- 通知済みでない

初期値は以下。

```text
3時間以上未完了
```

### 13.3 通知内容

```text
未完了タスクがあります：
「仕様書を確認する」
```

### 13.4 通知頻度

通知は控えめにする。

同じタスクに対する通知は、少なくとも数時間は再通知しない。

### 13.5 通知設定

将来的に以下を設定可能にしてもよい。

- 通知ON/OFF
- 通知までの時間
- 通知対象タスク数
- 通知音ON/OFF

## 14. 簡易メモ仕様

### 14.1 目的

タスクに短い補足情報を残せるようにする。

### 14.2 入力方法

タスクの詳細表示または展開表示でメモを入力する。

```text
メモ：
[ ここにメモを入力 ]
```

### 14.3 保存仕様

- memo に保存する
- 入力後、自動保存する
- updated_at を更新する

## 15. v1.5で追加するデータ項目

| 項目 | 型 | 必須 | 説明 |
|---|---|---|---|
| notified_at | datetime | 任意 | 通知日時 |
| archived | boolean | 任意 | 履歴化済みフラグ |
| memo | string | 任意 | 簡易メモ |
| last_viewed_at | datetime | 任意 | 最終表示日時 |

## 16. 画面構成

### 16.1 メイン画面

- タスク入力欄
- 未完了タスク一覧
- 完了済みタスク折りたたみ
- 今日の完了件数と作業時間
- 日報コピーボタン

### 16.2 履歴画面

- 今日
- 昨日
- 過去7日
- 日別完了タスク一覧
- 作業時間合計
- Markdownコピー

### 16.3 設定画面

- 起動時自動開始
- ショートカットキー表示
- 通知ON/OFF
- 通知時間設定

## 17. 非対象機能

v1.5では以下を実装しない。

- 右クリックメニューからタスク追加
- ブラウザ拡張連携
- 元URL保存
- 元アプリ名保存
- 複数ユーザー共有
- クラウド同期
- AI自動分類
- カレンダー連携

## 18. v1.5 完了条件

v1.5は、以下を満たした時点で完成とする。

- ショートカットキーでアプリを開ける
- 完了済みタスクを折りたためる
- タスク検索ができる
- 完了履歴を日付別に確認できる
- 今日の日報をMarkdown形式でコピーできる
- タスクタイトルを編集できる
- OS起動時の自動起動を設定できる
- 長時間未完了タスクを通知できる
- タスクに簡易メモを残せる

## 19. v1.5の価値

v1.5により、アプリは単なるToDoリストから、日々の作業ログツールへ進化する。

特に以下の価値が増える。

```text
・ショートカットで即起動できる
・完了履歴を振り返れる
・日報に転用できる
・作業時間の見える化ができる
・未完了タスクを見逃しにくくなる
```
